Underdark

Mobile peer-to-peer mesh networking library

Download this project as a .zip file Download this project as a tar.gz file

What is it?

Underdark is peer-to-peer mesh networking library for iOS and Android, with Wi-Fi and Bluetooth support.

Video demo: http://www.youtube.com/watch?v=ox4dh0s1XTw

Demo apps

  • iOS: http://itunes.apple.com/app/id956548749
  • Android: https://play.google.com/store/apps/details?id=me.solidarity.app

Please note that for Bluetooth connection to work on Android, one of the phones must have Android 5.x AND support Bluetooth LE Peripheral mode (have "yes" in the last column here: http://altbeacon.github.io/android-beacon-library/beacon-transmitter-devices.html).

Trial version

  • Android: http://github.com/virl/underdark-android
  • iOS version of the library will be available later.

Features

  1. Allows apps on mobile devices to auto-discover and connect to each other via Wi-Fi and Bluetooth without any intervention from the user and without using any Internet connection. Works both in foreground and background app modes.
  2. Allows asynchronous transfer of messages, photos and sounds — all without freezing the UI of the app;
  3. Routes messages from Wi-Fi to Bluetooth-connected devices and vice versa — so devices with turned off Wi-Fi will receive all the same data via Bluetooth.
  4. Work between Android<-->Android or between iOS<-->iOS via Bluetooth and Wi-Fi.
  5. Works between iOS<-->Android via infrastructure Wi-Fi only.
